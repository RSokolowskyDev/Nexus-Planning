# NexusPlan 🌌

**NexusPlan** is a high-fidelity, infinite-canvas planning application designed for seamless temporal navigation. Built with an architectural blueprint aesthetic, it offers a fluid, interactive experience for managing complex schedules across multiple scales.

![NexusPlan Preview](https://via.placeholder.com/1200x600.png?text=NexusPlan+Architectural+Canvas+Preview)

## ✨ Key Features

- **Infinite Planning Canvas**: Navigate through time without boundaries using intuitive pan and zoom gestures.
- **Semantic Multi-Scale Zoom**: Automatically transition between **Day, Week, and Month** views as you zoom in and out.
- **Blueprint Aesthetic**: High-contrast grid system featuring Gemini Spark intersections and precise dashed borders for a premium, technical feel.
- **Split-Screen Documentation**: A context-aware documentation system that allows for side-by-side editing of event details, notes, and metadata.
- **Mobile-Native Experience**: Specialized mobile interface with gesture support (swipe to change days, pinch to exit) and a native-feeling Day View overlay.
- **Dynamic Event Management**: Support for multiple event categories:
    - 📁 Projects
    - 🤝 Meetings
    - ✅ Tasks
    - 💎 Milestones
- **Smart Recurrence**: Integrated support for daily, weekly, and custom recurring events.
- **High-Performance Rendering**: Optimized with 2D frustum culling to ensure smooth interaction even with dense schedules.

## 🛠️ Technology Stack

- **Core**: Vanilla JavaScript (ESM)
- **Structure**: HTML5 Semantic Markup
- **Styling**: Vanilla CSS (Modern Design Tokens, Glassmorphism)
- **Typography**: [Outfit](https://fonts.google.com/specimen/Outfit) via Google Fonts
- **Icons**: Custom SVG implementation

## 🚀 Getting Started

NexusPlan is built as a lightweight, dependency-free static web application.

### Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/Nexus-Planning.git
   cd Nexus-Planning
   ```

2. **Run with a local server**:
   You can use any static file server. For example, using Node.js:
   ```bash
   npx serve .
   ```
   Or simply open `index.html` in your favorite modern web browser.

## 📂 Project Structure

- `index.html`: Main entry point and application skeleton.
- `main.js`: Core engine, state management, and rendering logic.
- `index.css`: Comprehensive design system and component styling.
- `README.md`: Project documentation.

## 🎨 Design Philosophy

NexusPlan prioritizes **visual excellence** and **interaction fluidity**. The "Architectural Blueprint" theme is achieved using high-contrast CSS gradients and custom SVG patterns, providing a productive yet aesthetically pleasing environment for long-term planning.
